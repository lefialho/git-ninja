ll ou ls -la = lista os arquivos do diretório

git log --name-status = mostra o log dos commits, juntamente com os arquivos que foram modificados, adicionados ou deletados

git diff --staged = mostra o que foi modificado no nível de staged

git diff (numero commit) (numero commit) = compara 2 commits

git rm arquivo = deleta o arquivo na staging area

git rm -rf node_modules/ --cached = Remove o node_modules da staging area quando adicionado acidentalmente, sem deletar o diretório real
